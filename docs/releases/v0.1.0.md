# Parallax v0.1.0

Parallax v0.1.0 is the first release of the project.

## Highlights

- Native Qt/QML desktop shell for a local-first cytometry workflow
- Shared Rust engine for deterministic gating and replay
- Rectangle and polygon gate authoring directly from scatter plots
- Undo and redo through explicit command history
- FCS inspection CLI
- Rust backend capability surface for parity groundwork

## Included Documentation

- Quick start
- User guide
- Tutorial
- Deployment guide
- Operations guide

## Known Gaps

- Desktop file import is not available yet
- Workspace save/load is not available yet
- Gate editing handles and plot pan/zoom are not available yet

## Release Focus

This release establishes the product shape and engineering guardrails:

- one engine everywhere
- local-first behavior
- replayable actions
- no silent automation
- performance-conscious desktop foundations
