# Parallax v0.2.0

Parallax v0.2.0 is the first feature release that moves the project from a demo shell toward a real local-first cytometry workstation.

## Highlights

- Added desktop import for one or many `.fcs` files with per-sample session state
- Added workspace save/load that reopens imported samples and replays saved history
- Added explicit analysis settings for parsed compensation and transform presets
- Added initial `Biexponential` and `Logicle` transform options in the shared Rust engine
- Added replayable plot-view controls for `Auto`, `Focus`, `Zoom In`, and `Zoom Out`
- Added an authentic public FCS regression suite and automated tests for the structured CLI inspection path

## Notes

- Undo and redo still apply to gate commands only
- Workspace persistence is still source-path based
- Histogram or density plots, batch templates, and reporting export are still ahead
