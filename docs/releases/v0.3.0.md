# Parallax v0.3.0

Parallax v0.3.0 turns the workstation milestone into a more usable multi-sample analysis tool for real lab review workflows.

## Highlights

- Added a native histogram panel and a shared Rust stats engine with counts, frequencies, means, and medians
- Added batch template application plus CSV export for active-sample stats and grouped batch stats
- Added selected-population comparison across loaded samples with explicit missing-population handling
- Added persisted cohort labels, cohort-summary cards, and cohort-summary CSV export
- Added replayable derived metrics for `Positive Fraction` and `Mean Ratio`, including workspace persistence, comparison deltas, cohort means, and derived-metric CSV export
- Expanded documentation and regression coverage for workspaces, comparisons, cohort summaries, and derived metrics

## Notes

- Undo and redo still apply to gate commands only
- Derived metrics are limited to the built-in positive-fraction and mean-ratio formulas today
- Workspace persistence is still source-path based
- Density plots, gate editing handles, and report-quality figure export are still ahead
