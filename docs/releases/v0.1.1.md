# Parallax v0.1.1

Parallax v0.1.1 is a polish patch release immediately following the initial public launch.

## Highlights

- Fixed desktop CI on GitHub Actions for Ninja-based runners
- Updated checkout actions to the current major version
- Kept the public release surface aligned with the shipped docs and workflows

## Why This Release Exists

The first public release established the product and documentation surface. This patch release tightens the release engineering path so the latest tag reflects a clean GitHub workflow state as well as a working desktop and backend codebase.
