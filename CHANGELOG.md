# Changelog

## v0.1.2 - 2026-03-20

Licensing and delivery-policy update.

- Declared the repository under Apache License 2.0
- Added license documentation to the README
- Updated the service delivery skill to remember Apache-2.0 as the default license for future public releases unless the user specifies otherwise

## v0.1.1 - 2026-03-20

Polish patch release for GitHub delivery and CI reliability.

### Highlights

- Fixed the Qt desktop CI build for Ninja-based GitHub runners by declaring the Rust bridge archive as a generated byproduct
- Updated GitHub Actions checkout steps to the current major release
- Published a cleaner release-ready metadata surface on GitHub

## v0.1.0 - 2026-03-20

Initial public release of Parallax.

### Highlights

- Shared Rust engine for deterministic gating and replay
- Native Qt/QML desktop shell
- Rectangle and polygon gating directly from the plot surface
- Command-log undo and redo
- FCS inspection CLI
- Rust backend capability surface for parity groundwork

### Documentation

- Added a quick start, user guide, and tutorial
- Added deployment, operations, and release documentation
- Added GitHub community health files and issue templates
