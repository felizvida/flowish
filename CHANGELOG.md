# Changelog

## v0.1.0 - 2026-03-20

Initial public release of Parallax.

### Highlights

- Shared Rust engine for deterministic gating and replay
- Native Qt/QML desktop shell
- Rectangle and polygon gating directly from the plot surface
- Command-log undo and redo
- FCS inspection CLI
- Rust backend capability surface for parity groundwork

### Documentation

- Added a quick start, user guide, and tutorial
- Added deployment, operations, and release documentation
- Added GitHub community health files and issue templates
